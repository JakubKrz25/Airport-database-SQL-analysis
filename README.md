# Airport database analysis

This project presents an analysis of an airport database using SQL techniques. The goal is to extract meaningful insights related to flight operations, passenger behavior, revenue and operational efficiency. The study is divided into 3 main parts: flights, passengers and revenue analysis.

The structure of this repository is as follows:
- in the catalog **SQL files** you can find raw SQL code for the three main parts of the analysis mentioned above.
- in the catalog **Analysis** you can find Jupyter notebooks with the analysis
  (SQL queries were executed via Python to leverage the structured and interactive environment of Jupyter Notebooks.)
- the file **EER diagram** contains the relational structure of the database.

**Remark**: This project is a work in progress and is currently in its early stages (especially Passengers and Revenue analysis). Additional analyses and insights will be added.
  
This project uses a publicly available database at https://dev.mysql.com/doc/airportdb/en/airportdb-introduction.html.
